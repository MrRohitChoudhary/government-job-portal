JAVASCRIPT ENHANCEMENTS
═══════════════════════════════════════════════════════════════════════════════

Added dynamic category loading:

  fetch('/api/categories')
    .then(r => r.json())
    .then(cats => {
        const menu = document.getElementById('categoryMenu');
        if (menu && cats && cats.length > 0) {
            menu.innerHTML = '';
            cats.forEach(c => {
                const li = document.createElement('li');
                const a = document.createElement('a');
                a.href = '/jobs?category=' + c.id;
                a.textContent = c.name;
                a.title = c.name;
                li.appendChild(a);
                menu.appendChild(li);
            });
        }
    })

This ensures:
• Categories load automatically when page loads
• Each category links to filtered jobs page (/jobs?category=ID)
• Graceful error handling with console logging
